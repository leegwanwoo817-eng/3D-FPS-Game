# 3D FPS Game

A beginner-friendly 3D first-person shooter game made with Unity.

## Project Structure

```
Assets/
├── Scripts/
│   ├── PlayerController.cs      # Handles player movement and shooting
│   ├── EnemyAI.cs              # Simple enemy behavior
│   ├── GameManager.cs          # Manages game state
│   └── HealthSystem.cs         # Health and damage system
├── Scenes/
│   └── MainScene.unity         # Main gameplay scene
└── Prefabs/
    ├── Enemy.prefab            # Enemy prefab
    └── Bullet.prefab           # Bullet prefab
```

## Getting Started

1. **Install Unity** (2021 LTS or newer)
2. **Clone this repository**
3. **Open in Unity** - File → Open Project
4. **Open MainScene** in Assets/Scenes/
5. **Press Play** to test!

## Features

- ✅ Player movement (WASD)
- ✅ Mouse look camera
- ✅ Shooting mechanics (Left Click)
- ✅ Basic enemy AI
- ✅ Health and damage system
- ✅ Simple UI

## Controls

- **W/A/S/D** - Move
- **Mouse** - Look around
- **Left Click** - Shoot
- **ESC** - Pause (optional)
